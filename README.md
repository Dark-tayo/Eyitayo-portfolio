SkillsForge Frontend Internship - Personal Portfolio
This repository contains my submission for the SkillsForge Second Stage Selection task (Frontend Track). It is a responsive personal portfolio built to showcase my profile, background, and technical interests.

Requirements Fulfilled
This project meets all the specified criteria for Task A: Personal Portfolio:
Home Section: Includes a clear profile image, full name, tagline, and key interests.
About Section: An extended bio detailing my background and motivations with a minimum of 3 paragraphs.
Experience Section: Highlights my educational background and project experience.
Social Links: Clickable official icons for social media platforms.
Contact Card: A dedicated section for my Email, Phone number, and relevant contact info.

 Bonus Features Implemented
Dark/Light Mode: A toggle to switch between themes.
Responsive Layout: The site works seamlessly on both mobile and desktop screens.
Smooth Scrolling: Simple animations for a better user experience.
Clean Design: A visually appealing and simple layout.

Built With
HTML5
CSS3 (Custom Properties for themes)
JavaScript (Vanilla)
Font Awesome (For official brand icons)
